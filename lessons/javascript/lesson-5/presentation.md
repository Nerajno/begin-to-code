## Hide and Show

To hide an element:
```js
$(...).hide();
```

To show a hidden element:
```js
$(...).show();
```

---
To hide and show an element, you first get the element out from the web page
using a CSS selector as you've done before, and then call either the `hide` or
the `show` method.
